# Plugin API

## game module

- `spawn_npc`
- `teleport_peer`
- `set_weather`
- `show_popup`
- `get_peer_positions`
- `dist`
- `spawn_vehicle`
- `send_rpc`
- `register_packet`
- `register_panel`
- Each plugin bundle must include `manifest.json` specifying allowed RPC names
  under `rpc_allow`.
